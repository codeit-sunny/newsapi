# 📰 News API (Django REST Framework)

A simple and scalable News API built using Django REST Framework.  
This project provides CRUD operations for managing news articles.

----------------------------------------------------------------------

## 🚀 Features

- ✅ Create News
- 📖 Read / Fetch News
- ✏️ Update News
- ❌ Delete News
- 🔍 API-based data handling (JSON)
- ⚡ Fast & scalable backend

----------------------------------------------------------------------

## 🛠️ Tech Stack

- Python 🐍
- Django
- Django REST Framework (DRF)
